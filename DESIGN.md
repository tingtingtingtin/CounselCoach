## Overview

Healthcare interface with warm, optimistic energy. The design balances clinical trust with human warmth through generous spacing, soft rounded corners, and a carefully curated color palette that avoids sterile medical aesthetics.

Generous whitespace with 48px section spacing as the primary rhythm. Content contained within max-width containers (1100px) with responsive grid systems. Consistent 24px padding for cards and components.

Uses a 8px base grid with scale: 1, 1.5, 2, 3, 4, 6, 10.

## Colors
- **Primary Yellow** (#ffc700): Primary CTA buttons, brand accents, highlights
- **Forest Dark** (#123333): Primary text, headings, high contrast elements
- **Forest Medium** (#446464): Secondary text, navigation, subtle elements
- **Forest Light** (#7f9595): Borders, dividers, inactive states
- **Smoke Gray** (#e5e7eb): Section backgrounds, cards, subtle containers
- **Pure White** (#ffffff): Primary background, cards, overlays
- **Mint Green** (#9be8a8): Success states, positive feedback
- **Sky Blue** (#9fe3f9): Information states, secondary accents
- **Clay Red** (#fe9a9a): Error states, warnings, alerts

## Typography
- **Headline Font**: Uncut Sans Variable
- **Body Font**: Uncut Sans Variable
- **Label Font**: Uncut Sans Variable

Uncut Sans Variable serves all typographic roles with weight differentiation creating hierarchy. Display text uses lighter weights (400) with negative letter-spacing for elegance, while UI elements employ semibold (600) weights for clarity. The scale progresses logically from 14px labels to 56px heroes, with line-heights optimized for readability at each size.

## Elevation

Minimal shadow strategy with subtle elevation on cards and modals. Primary depth is created through color contrast and layering rather than dramatic shadows, maintaining the clean, accessible aesthetic.

## Components
- **Primary CTA Button**: Yellow background (#ffc700) with rounded corners (24px), medium padding, and semibold text
- **Secondary Button**: Outlined button with forest green border, transparent background, and forest green text
- **Feature Card**: White background with 24px border radius, generous padding, and subtle shadow
- **Navigation Bar**: Clean horizontal navigation with forest green text and yellow accent for active states
- **Hero Section**: Large background image with overlay text, prominent heading, and dual CTA buttons
- **Privacy Modal**: White overlay with rounded corners, clear typography hierarchy, and toggle switches

## Do's and Don'ts
- Do use the yellow primary color sparingly for maximum impact on CTAs and key actions
- Don't mix border radius values - stick to the defined scale for consistency
- Do maintain generous whitespace with 48px as the primary section spacing unit
- Don't use font weights below 400 or above 700 to maintain readability
- Do ensure sufficient contrast ratios with the forest green text colors
- Don't overuse accent colors - reserve mint, sky, and clay for semantic purposes only
